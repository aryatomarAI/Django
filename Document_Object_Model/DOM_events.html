DOM Events

We don't always want to have to specify beforehand how to interact with the DOM.

Many times we only want the interaction to occur on a particular event, such as a click or a hover. we can achieve this by adding an event Listener.

Listening for an event looks like this: myvariable.addEventListener(event,function)

There are many possible events like:

  • clicks
  • hovers
  • Double clicks
  • Drags
